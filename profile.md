# 提示词生成系统

## 概述

该 web 应用根据场景不同，通过填表单的方式生成提示词。两大核心功能：

1. 帮助用户完善思考
2. 结构化提示词。

## 技术栈

前端技术栈：

1. vite + vue3 + element-plus + typescript + axios + pinia + vue-router + less
后端技术栈：
1. 后端框架：Flask (Python)
1. 数据库：mysql+SQLite

### 全局

- 提示词技巧
  1. 技巧一，点击复制按钮，生成结果中添加文案
  2. 技巧二，作为参考常驻页面，支持缩放
- 生成提示词被复制之后，弹出一个表单，表单包含复盘检查清单
  1. 预期达到的效果？
  2. 如何评价（验证）这次生成的结果？
  3. 是否有明显错误答案？你怎么处理的？
  4. 生成的内容和你的预期不符，我是如何调整优化的？
  5. 我为什么这么写提示词？

AI_workflow 是我写的文档，我现在想基于这篇文档开发一个web应用，通过vercel部署。
目前有四个页面：

- 首页
- 最小公式
- 提示词模版列表
- 复杂提示词
